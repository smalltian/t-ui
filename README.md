# zhujuntian-ui

## 安装

```
npm run zhujuntian-ui or yarn zhujuntian-ui

```

## 使用

```
import zhujuntianui from 'zhujuntian-ui'
Vue.use(zhujuntianui)

<template>
  <v-message message=" hello ,欢迎使用zhujuntian-ui组件库"></v-message>
</template>
<script>
  export default {
    data () {
      return {
      }
    }
  }
</script>
```
